# DarkFaceDetect
Face Detect in Dark Image With [MIRNet](https://github.com/soumik12345/MIRNet) And [Haar-cascade](https://docs.opencv.org/4.5.1/db/d28/tutorial_cascade_classifier.html)<br><br>
